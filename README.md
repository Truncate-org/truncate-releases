# Truncate IDE — Public Releases

This repository is the automated distribution point for the Truncate IDE. It hosts the latest signed binaries, application bundles, and update manifests.

> [!NOTE]
> This is a read-only distribution mirror. For the source code, issue tracking, and documentation, please visit the [Main Truncate Repository](https://github.com/Truncate-org/truncate-ide).

---

## Installation

To install the latest version, go to the [Releases](https://github.com/Truncate-org/truncate-releases/releases/latest) page and download the package for your platform:

### macOS (Apple Silicon & Intel)
* Download the .dmg file.
* Drag Truncate to your Applications folder.

### Linux (Ubuntu/Debian)
* Download the .deb file.
* Install via: sudo dpkg -i truncate_*.deb

### Linux (Arch/Universal)
* Download the .AppImage.
* Make it executable: chmod +x truncate_*.AppImage
* Run directly.

---

## Auto-Updates
Truncate features built-in secure auto-updates. All binaries in this repository are cryptographically signed. The application will automatically check this repository for new releases and prompt you to update when a new version is verified.
